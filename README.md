# 🔦⚡Laser Tag⚡🔦



**📌 Project Overview**



A Multiplayer Game of ***Laser Tag***. Where players shoot each other with a gun that shoots lasers to kill the other player. It makes use of the **ESP NOW** Protocol to make a Peer-to-peer architecture, making a gaming platform between that runs between the network of Gun Modules. 

The gun uses IR signals as the communication between the devices when one player is hit. We align the IR Emitter with the Laser module to ensure the visual point of impact (through the Laser) and the digital signal area is aligned.
